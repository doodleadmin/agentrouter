# Agents Memory — Профили агентов

## Описание

Профили агентов в memory vault. Runtime-профили агентов (system prompts, permissions) находятся в `.opencode/agents/`.

## Связь с runtime

- `.opencode/agents/<slug>.md` — runtime-профиль (system prompt, permissions)
- `.ai_memory/agents/<slug>.md` — memory-профиль (история, заметки, наблюдения)
