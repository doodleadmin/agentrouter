# Шаблон: Architecture Decision Record (ADR)

**Инструкция:** При принятии архитектурного решения создать файл `.ai_memory/decisions/<NNNN>-<title>.md`. Нумерация сквозная.

---

```markdown
# <NNNN>: <Название решения>

Дата: <YYYY-MM-DD>
Статус: proposed | accepted | deprecated | superseded
Автор: <agent_slug>

---

## Контекст
<Ситуация, требования, ограничения>

## Решение
<Что решено сделать, чётко и конкретно>

## Альтернативы

### Вариант A: <Название>
- Плюсы: ...
- Минусы: ...

### Вариант B: <Название>
- Плюсы: ...
- Минусы: ...

## Последствия

### Положительные
- ...

### Отрицательные
- ...

## Затронутые компоненты
- <Компонент>

## Ссылки
- <Ссылка>
```
