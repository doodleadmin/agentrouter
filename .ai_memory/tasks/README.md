# Tasks — Логи выполненных задач

## Описание

Summary каждой выполненной задачи записывается в отдельный markdown-файл.

## Структура

```
.ai_memory/tasks/
└── <date>-<task-id>.md
```

Примеры:
- `2026-05-03-task-0001.md`
- `2026-05-04-task-0002.md`

## Шаблон

См. [../templates/task-summary-template.md](../templates/task-summary-template.md)
