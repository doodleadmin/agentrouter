## Что было исправлено

### 1. `[tool]` → `[build-system]` (pyproject.toml)
Корневая проблема: PEP 517/518 требует `[build-system]`, а не `[tool]` для указания бэкенда сборки.

### 2. Добавлен hatch build config (pyproject.toml)
```toml
[tool.hatch.build]
packages = ["app"]

[tool.hatch.build.targets.wheel]
packages = ["app"]

[tool.hatch.build.targets.sdist]
include = ["app", "alembic", "alembic.ini", "pyproject.toml"]
```
Явно указали, что Python-пакет — только `app` (а `alembic/` — служебный).

### 3. `metadata` → `chunk_metadata` (memory_chunk.py)
`metadata` зарезервирован в SQLAlchemy Declarative API. Переименован Python-атрибут с сохранением имени колонки в БД через `mapped_column("metadata", ...)`.
